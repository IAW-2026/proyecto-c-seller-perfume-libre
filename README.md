# seller

Aplicación **Seller** del [Proyecto IAW 2026](https://iaw-2026.github.io/proyecto/) — comisión `Perfume Libre`.

Esta app corresponde al rol del vendedor en los proyectos de tipo **B (Delivery)** y **C (Marketplace)**.

Con esta aplicacion un usuario podra crear publicaciones y manejarlas de forma sencilla, ya que se divide la atencion en la parte Mis productos que se enfoca en el manejo de los productos publicados y por otro lado se tienen las Reseñas que compacta la toda la informacion de manera simple para ver la satisfaccion del cliente.
Cuando una persona compre un producto, este le aparecera en el apartado ordenes, en donde se puede confirmar que el paquete ya esta hecho para que un repartidor lo venga a buscar.
Si un producto llega a tener stock cero, este se considera en estado pausado y el vendedor decidira si quiere seguir vendiendo el producto agregando mas stock a la publicacion o borrar la publicacion para que ya no le aparezca en Mis productos.
